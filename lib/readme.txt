3rd party library usage


JGIT - git api - used for syncing user library with GIT
SL4J - Logging library, used by JGIT
JSCH - ssh security, used by JGIT
flatlaf - seb import for more modern UI
flatlaf-intellij-themes - seb import for more modern UI
commons-lang3 - for StringUtils used to abbreviate long file paths
