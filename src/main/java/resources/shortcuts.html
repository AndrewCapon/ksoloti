Ksoloti Patcher Keyboard Shortcuts

Note: on MacOS use the CMD key wherever the CTRL key is used in this document.

Patch Window

Key Combo

Function

Notes
CTRL + X Cut Cuts the selected objects including their interconnections and places them in the clipboard.
CTRL + C Copy Copies the selected objects including their interconnections to the clipboard.
CTRL + V Paste Pastes the content of the clipboard.
DEL Delete Deletes the currently selected objects.
CTRL + A Select All Selects all objects in the current patch. Useful if you want to copy everything to a subpatch or realign all objects in your patch window.
CTRL + Z Undo Reverts the last operation. (I believe currently not all operations are covered, e.g. comments or changing object labels?)
CTRL + Y, CTRL + SHIFT + Z Redo Reverts the last undo, in other words re-does the last operation. (I believe currently not all operations are covered, e.g. comments or changing object labels?)
CTRL + S Save Reverts the last undo, in other words re-does the last operation. (I believe currently not all operations are covered, e.g. comments or changing object labels?)
CTRL + SHIFT + S Save As... Opens a file selection dialog to save the current patch as a new file.
CTRL + W Close Closes the current patch window.
TAB, SHIFT + TAB Cycle Focus Cycles through the currently selected attribute or parameter.
CTRL + E Toggle Live Mode Takes your patch live or brings it back to edit mode.
CTRL + I Open Patch Settings Opens the current patch's Settings window where its receiving MIDI channel, subpatch mode and other options can be changed.
CTRL + T Open Patch Notes Opens the current patch's Notes where you can save notes about your patch.
CTRL + B Toggle Patch Cords in Background Toggles patch cord visibility either on top or hidden by the objects. Convenient in complex patches or when you're done editing and want to have a cleaner canvas.
SPACE Show Object Finder Opens the Object Finder that lets you search for and add objects to your patch.
